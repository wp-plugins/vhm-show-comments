Show comments on your pages, posts, sidebar with a shortcode or PHP code.

== Description ==

An easy and straight to the point plugin to show comments on your pages, posts, sidebar with a shortcodes or PHP code.

Major features in VHM Show Comments includes:

* Fully customizable template
* Shortcode and PHP function with custom parameters to display on any page, post, text widget or PHP file

== Installation ==

1. Upload the VHM Show Comments plugin to your blog
1. Activate the plugin
1. Go to `Settings > VHM Show Comments` for further information
