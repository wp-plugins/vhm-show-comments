Show comments from pages, posts, categories, tags or user comment.

== Description ==

An easy and straight to the point plugin to generate shortcodes to show comments on your pages, posts or text widgets from the following sources:

* A page or post
* A category or tag
* A comment from a user

Major features in VHM Show Comments includes:

* Fully customizable template via variables
* Shortcode or PHP function to display on any page, post, text widget or PHP file
* Ascendant, descendant or random order.

== Installation ==

1. Upload the VHM Show Comments plugin to your blog
1. Activate the plugin
1. Go to `Comments > VHM Show Comments` for further information
