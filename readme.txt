Get a list of comments (or a single comment providing its ID) and show them on a page, post or sidebar using a shortcode or a PHP code.

== Description ==

An easy and straight to the point plugin to show comments on your pages, posts OR sidebar with a shortcode or PHP code.

Major features in VHM Show Comments includes:

* Fully customizable template
* Shortcode and PHP function with custom parameters to display on any page, post, text widget or PHP file
* Order comments ascendant, descendant or random.

== Installation ==

1. Upload the VHM Show Comments plugin to your blog
1. Activate the plugin
1. Go to `Settings > VHM Show Comments` for further information
